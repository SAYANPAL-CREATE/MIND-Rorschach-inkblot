# MIND-Rorschach Dual Pipeline Portal

This merged website contains two distinct prototype pipelines:

1. Mental Health Questionnaire
2. Rorschach Inkblot Test

Both pipelines generate patient-friendly results, reasons behind the output, and downloadable PDF reports.

## Important Disclaimer

This is an educational and experimental digital-health prototype. It is not a clinical diagnosis, not a validated Rorschach scoring system, and not a substitute for assessment by a licensed mental-health professional.

## Files

- index.html
- README.md

## Run Locally

Open `index.html` directly in a browser or use VS Code Live Server.

## GitHub Pages Deployment

Upload `index.html` to the root of your GitHub repository.

Then enable:

Settings → Pages → Deploy from branch → main → /root
